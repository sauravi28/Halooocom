HalooCONNECT FAQ

  1. What is the purpose of this manual?

    This manual serves as a resource for Call Center Managers and System Administrators to understand how the HalooCONNECT call center solution works and learn how to use its features properly.

  2. How do I access the HalooCONNECT admin page?

    Enter the IP address of the server (e.g., 10.9.6.3) in a web browser. Then click on the "Admin" option and enter the admin username and password.

  3. How do I create a new user?

    Click on "Users" menu > "Add a new user". Enter the username, password, and set the user level (6 for agents, 9 for administrators). Optionally, enter phone login and password for extension login.

  4. How do I modify user settings?

    Click on the user ID you want to modify, then make changes like updating passwords or enabling/disabling features like manual dial, scheduled callback, hotkeys, etc. Click "Submit" to save changes.

  5. How do I delete a user?

    Click on the username you want to delete, then click "DELETE THIS USER" and confirm deletion.

  6. How do I copy user settings to a new user?

    Click "Copy User", enter the new username and password. The other settings will be copied from the source user.

  7. How do I view user activity statistics?

    Go to "Users" menu > "User Stats". Select the start/end date and username to generate the report.

  8. How do I modify a campaign?

    Click on the "Campaign ID" you want to modify, then click "Detail" to edit campaign options.

  9. How do I delete a campaign?

    Select the campaign, scroll down, click "DELETE THIS CAMPAIGN", then confirm deletion.

  10. How do I copy a campaign?

    Click "Copy Campaign" to create a new campaign with the same settings as the source campaign (leads are not copied).

  11. What is a "Hopper" in a campaign?

    The Hopper is a buffer that loads a pre-set number of leads from the database for outbound dialing.

  12. What are Statuses and how do I add them?

    Statuses are dispositions that agents select during call wrap-up. Go to "Campaigns" menu > "Statuses" to add new dispositions.

  13. What are Pause Codes and how do I add them?

    Pause Codes are reasons for agents to go on pause (tea break, lunch, etc.). Go to "Campaigns" menu > "Pause Codes" to add new codes.

  14. What are Hotkeys and how do I configure them?

    Hotkeys provide keyboard shortcuts for quick call disposition. Go to "Campaigns" menu > "Hotkeys" to enable/configure them.

  15. How do I recycle leads with specific dispositions?

    Go to "Campaigns" menu > "Campaign Lead Recycling". Select the dispositions to recycle, enable "Reset Hopper", then activate recycling in the campaign list.

  16. What is Auto Alt-Dialing?

    If enabled, the server will dial an alternate number (if available) for a lead when getting dispositions like busy, no answer, etc.

  17. What is List Mix and how does it work?

    List Mix allows dialing of leads from two lists together randomly or evenly within the same campaign based on configurable settings.

  18. How do I upload lead lists?

    Go to "Lists" menu > "Add A New List", assign list ID and campaign. Then "Load New Leads" to browse and upload the lead file (.csv or tab delimited).

  19. How do I download a lead list?

    Go to "Lists" menu > "Show List", select the list, then click "Click here to download this list".

  20. How do I add scripts for agents?

    Go to "Scripts" menu, enter the script text which will pop up for agents on receiving calls. Customer data can be highlighted in scripts.

  21. What are In-Groups and how are they configured?

    In-Groups are inbound call queues that agents log into to receive calls from specific DID numbers. Configure welcome message, music on hold, etc.

  22. How do I add a new DID number?

    Go to "Inbound" menu > "Add A New DID". You can route incoming DID calls to IVR, In-Group queues, or extensions.

  23. What is a Call Menu and how do I configure IVR?

    A Call Menu is an IVR flow configured with voice prompts and routing based on caller DTMF inputs. Create and modify IVRs under "Inbound" > "Call Menu".

  24. What are User Groups and their purpose?

    User Groups allow splitting agents into different groups with customizable permissions, campaign access restrictions, and shift timings.

  25. How do I view and manage phone extensions?

    Go to "Admin" menu > "Phones" to view, add or modify phone extensions that agents will use to log in.

  26. What are Remote Agents?

    Remote Agents are virtual agents used for voice broadcast services where pre-recorded voice messages are played when calls connect.

  27. How do I manage system-wide call dispositions?

    Go to "Admin" menu > "System Statuses" to enable/disable or add new global call disposition codes.

  28. How do I configure servers?

    The "Admin" > "Servers" menu allows viewing, adding and modifying the settings of telephony servers like Asterisk version, etc.

  29. How do I upload voice prompts and recordings?

    Use the "Admin" > "Audio Store" menu to upload voice prompt files in 16bit 8kHz PCM WAV format for IVR, music on hold, etc.

  30. How do I configure music on hold?

    In "Admin" > "Music On Hold (MOH)", you can select audio files to play in random or sequential order when callers are held.

  31. How do I access real-time reports and monitoring?

    The "Reports" menu provides access to real-time agent monitoring, live call listening/barging, service level reports, and agent performance details.

  32. How do I download call recordings?

    Go to "Admin" > "Voice Logger" to filter, play and download the recorded call audio files from the server.
