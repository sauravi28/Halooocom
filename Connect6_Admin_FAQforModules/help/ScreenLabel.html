• Label ID*

This field needs to be at least 2 characters in length and no more than 20 characters in length, no spaces or special characters. This is the ID that will be used to identify the screen label throughout the system.

• Label Name*

This is the descriptive name of the screen label entry.

• Label Active*

Defines whether the list needs to active or not.

• Label Phone Number

Will showcase the customer contact number here (field is hardcoded as calling number is mandatory for each & every call)

• Label Comments

Filed to enter the call remarks or to pass on the feedback of the call

• Add Textbox

Will allow the admin to add extra fields based on the process requirements. Field infor can be either upload by the admin or will be added by the user during a live call.

• Add Dropdown

Will allow the admin to add dropdown option under first field admin needs to enter the dropdown header & on the below fields admin needs to enter the options for that particular dropdown separated by "," sign. i.e. connected,not connected,switchedoff
