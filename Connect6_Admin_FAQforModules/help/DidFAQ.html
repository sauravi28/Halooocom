• DID

This is the number, extension or DID that will trigger this entry and that you will route within the system using this function. There is a reserved default DID that you can use which is just the word -default- without the dashes, that will allow you to send any call that does not match any other existing patterns to the default DID.

• DID ASSIGN TO*

This the type of route that you set the DID to use
Ingroup - will send calls directly to the specified inbound group
IVR - Will send the call to the inbound IVR created.
