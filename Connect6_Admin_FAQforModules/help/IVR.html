• IVR

This option can be utilized for both Inbound calls & Robo call ( Voice blast ).

1. Click on Add new IVR.
2. IVR Name - Name of the IVR.
3. IVR Description - Full description on the IVR.
4. Choose number of levels - Number of levels the IVR needs to have ( 0 - 9 ).
5. Choose an option:
(i) Upload File - To upload a voicefile if already available. (ii) Text to Speech - Online TTS, to enter the text details which will be converted to voice.

• Choose Audio type

To select the Suitable Voice. - Then Convert. By doing so the IVR Mail file will be created later we can add on sub & Sub sub level by clicking on "+" option. Later this IVR can be assigned to a Ingroup or a Voice Blast Campaign.
