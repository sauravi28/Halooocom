• User ID*

Unique ID for the user to login ( Alphanumeric characters )

• User Name*

User full name

• User Password*

Login Password ( 8-16 letter special alphanumeric characters )

• User Level*

Levels based on user profiles Agent - Calling, TL - Dashboard & report download, Admin - Creation & modification, Super Admin - Complete overall access"

• User Status*

Defines the user ID current state Active - Will be able to login, Inactive - Will not be able to access the application

• Extension Id

ID mapped to a users for voice communication access ( Configured under softphones i.e. Zoiper, X-lite, linphone )

• Extension Password

Extension password for registration

• Assign Campaign

Campaign allocation for the User id limiting the access for the user.

• Work Type*

Work from office - Will have softphone calling access ( 1 sim required for calling )
Work from home - Will have two-leg calling which includes Agent sim card number + 2 sim card for a single call )
